# ESCO技能能力评分系统 - 执行计划

## 目标
根据职位描述和学历要求，对每个二级能力维度进行0和1-5的量化评分：
- **0**: 该岗位不涉及该能力维度
- **1-5**: 涉及该能力，水平从低到高递增

---

## 二、学历要求对能力等级的影响

学历作为调整因子，影响能力等级的基准判定：

| 学历 | 基准等级调整 | 说明 |
|------|------------|------|
| 不限/中专/高中 | -0.5 | 降低能力等级判定 |
| 大专 | 0 | 标准基准 |
| 本科 | +0.5 | 提高能力等级判定 |
| 硕士 | +1.0 | 显著提高能力等级 |
| 博士 | +1.5 | 最高能力等级判定 |

**应用方式**：
- 当职位描述判定为Level 2，学历为本科时，最终等级可能上调至Level 3
- 当职位描述判定为Level 3，学历为中专时，最终等级可能保持在Level 2-3之间

#### 步骤2A：关键技能词翻译方案

```python
# 构建中英翻译字典
translation_dict = {
    '沟通': 'communication',
    '协作': 'collaboration', 
    '管理': 'management',
    'Office': 'Microsoft Office',
    '数据分析': 'data analysis',
    '编程': 'programming',
    # ... 扩展至500-1000个常用技能词
}

def translate_description(chinese_desc):
    """将中文职位描述中的关键技能词翻译为英文"""
    translated = chinese_desc
    for cn, en in translation_dict.items():
        if cn in translated:
            translated += f" {en}"  # 追加英文词
    return translated

# 构建三层映射关系
skill_mapping = {
    'communication': {
        'esco_uri': 'http://data.europa.eu/esco/skill/...',
        'cn_keywords': ['沟通', '交流', '表达', '汇报', '协调'],
        'en_keywords': ['communication', 'interact', 'express'],
        'level_indicators': {
            1: ['协助沟通', '配合'],
            2: ['日常沟通', '接待'],
            3: ['独立沟通', '对接'],
            4: ['商务谈判', '关系管理'],
            5: ['战略合作', '高层对接']
        }
    },
    'digital_technology': {
        'esco_uri': 'http://data.europa.eu/esco/skill/...',
        'cn_keywords': ['Office', 'Excel', 'Word', 'PPT', '软件', '电脑'],
        'en_keywords': ['Microsoft Office', 'software', 'computer'],
        'level_indicators': {
            1: ['基本', '了解'],
            2: ['熟练', '掌握'],
            3: ['精通', '专业'],
            4: ['开发', '设计'],
            5: ['架构', '专家']
        }
    },
    # ... 为22个能力维度建立映射
}

class HybridESCOEvaluator:
    """混合ESCO评估器：中文关键词 + ESCO标准"""
    
    def __init__(self, use_extractor=False):
        self.skill_mapping = skill_mapping
        self.use_extractor = use_extractor
        if use_extractor:
            self.extractor = SkillExtractor()
    
    def evaluate_job(self, description, education):
        results = {}
        
        # 方法1：中文关键词匹配
        for skill_name, skill_info in self.skill_mapping.items():
            score = self._match_chinese_keywords(
                description, 
                skill_info
            )
            
            # 方法2（可选）：ESCO工具增强验证
            if self.use_extractor and score > 0:
                translated = self._translate_key_terms(
                    description, 
                    skill_info['cn_keywords']
                )
                esco_result = self.extractor.get_skills([translated])
                if skill_info['esco_uri'] in esco_result[0]:
                    score = min(5, score + 0.5)  # 提升置信度
            
            # 学历调整
            score = self._adjust_by_education(score, education)
            results[skill_name] = int(score)
        
        return results
    
    def _match_chinese_keywords(self, desc, skill_info):
        """基于中文关键词匹配和等级判定"""
        # 判断是否涉及
        cn_keywords = skill_info['cn_keywords']
        if not any(kw in desc for kw in cn_keywords):
            return 0
        
        # 判定等级
        for level in [5, 4, 3, 2, 1]:
            level_kws = skill_info['level_indicators'].get(level, [])
            if any(kw in desc for kw in level_kws):
                return level
        
        return 2  # 默认中级
```

## 六、风险与应对

### 技术风险

| 风险 | 影响 | 应对措施 |
|------|------|---------|
| esco-skill-extractor不支持中文 | 高 | 准备方案B和C作为备选 |
| 关键词匹配不准确 | 中 | 建立同义词库，增加验证样本 |
| AI模型处理速度慢 | 中 | 分批处理，使用GPU加速 |
| 职位描述质量差 | 低 | 预处理文本，清洗噪声 |

### 数据质量风险

| 风险 | 缓解措施 |
|------|---------|
| 职位描述过短或不完整 | 设置最小字符数阈值 |
| 包含大量HTML标签或特殊字符 | 文本预处理清洗 |
| 学历要求格式不统一 | 标准化学历字段 |

### 评分准确性限制

1. **基于规则的方法局限**：无法理解复杂语义和上下文
2. **隐含能力识别困难**：如"5年经验"隐含的能力水平
3. **行业差异**：同一技能在不同行业要求不同
4. **时间因素**：2016年的技能需求与现在可能不同

### 持续优化方向

1. 建立专家验证机制
2. 收集典型岗位作为标准案例
3. 根据反馈迭代关键词库
4. 考虑引入机器学习模型（长期）
