## 🛠️ 过滤工具

### 1. 基础过滤器 (`taskpair_filter.py`)
- **功能**: 过滤已知固定搭配和不合理组合
- **方法**: 规则-based词典匹配
- **适用**: 快速移除明显固定搭配

### 2. 智能过滤器 (`smart_filter.py`)
- **功能**: 基于语义连贯性和频率分布的智能过滤
- **方法**: 词性分析 + 语义评分 + 统计过滤
- **适用**: 深度优化任务对质量

### 使用方法
```bash
# 基础过滤
python taskpair_filter.py

# 智能过滤
python smart_filter.py
```

## 📈 处理流程

1. **文本清洗**：移除HTML标签、特殊字符、多余空格
2. **分词标注**：使用jieba进行中文分词和词性标注
3. **任务对提取**：
   - 动词 + 名词 → 任务对
   - 名词 + 动词 → 调换顺序后作为任务对
4. **频率统计**：统计动词、名词、任务对的出现频率
5. **结果过滤**：过滤低频项目（默认min_freq=5）

## 🎯 核心功能

- **动词识别**：`{'v', 'vn', 'vd', 'vg'}` 动词相关词性
- **名词识别**：`{'n', 'nr', 'ns', 'nt', 'nz', 'nl', 'ng'}` 名词相关词性
- **停用词过滤**：内置常用停用词列表
- **内存优化**：分批处理，避免内存溢出
- **进度监控**：实时显示处理进度和性能
